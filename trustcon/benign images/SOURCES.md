# Benign demo images

Harmless images used to populate the workshop's benign HMA hash bank ("CCF
known-content stand-in (benign)") so hash matches show up in the demo, in place
of a real TVEC hash set. Loaded with `npm run seed-hash-bank-images`.

All files are from Wikimedia Commons; the filename is the Commons page title.
Look up `https://commons.wikimedia.org/wiki/File:<filename>` for each file's
author and license (CC BY / CC BY-SA / public domain), and keep attribution if
you reuse them outside this workshop.
